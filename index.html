metaMediaViewer - Visualize multimedia elements.

A simple component for angular applications. It allows you to view a carousel of elements. Compatible media : images, video, audio, pdf.

Advanced configuration for Pdf viewer

By default the location of PDF.js assets are automatically determined. However if you place them on alternative locations they may not be found. If so, you can configure these locations.

You may choice to disable use of Web Workers API. This is useful if you want to add pdf.worker.js to your concatinated JavaScript file. However this will have a negative effect on the runtime performance.

To reference the viewer.css file, override the pdf.js-viewer options in your bower.json like this:
